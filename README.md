# 🌿 Kare Ons Herbal

Kare Ons Herbal is a modern Herbal & Ayurvedic E-commerce website built with Laravel. The platform allows customers to discover, purchase, and learn about premium herbal products while providing an easy-to-manage admin dashboard for inventory, orders, blogs, and customer management.

# 📦 Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/kare-ons-herbal.git
```

---

## Move into Project

```bash
cd kare-ons-herbal
```

---

## Install Dependencies

```bash
composer install

npm install
```

---

## Create Environment File

```bash
cp .env.example .env
```

---

## Generate Application Key

```bash
php artisan key:generate
```

---

## Configure Database

Update your `.env`

```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=kareons
DB_USERNAME=root
DB_PASSWORD=
```

---

## Run Migration

```bash
php artisan migrate
```

If seeders are available

```bash
php artisan db:seed
```

---

## Build Frontend

Development

```bash
npm run dev
```

Production

```bash
npm run build
```

---

## Start Server

```bash
php artisan serve
```

Open

```
http://127.0.0.1:8000
```
